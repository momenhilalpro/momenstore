## 💡 Quick Reference

### If Menu Doesn't Show
1. Check screen width (should be ≤768px)
2. Hard refresh (Ctrl+Shift+R)
3. Check DevTools
4. Try different browser

### If Menu Doesn't Work
1. Clear browser cache
2. Reload page
3. Try real device (not emulator)
4. Check console for errors (F12)

### If Navigation Doesn't Work
1. Verify all HTML href links are correct
2. Check all page files exist
3. Hard refresh page
4. Check network in DevTools
